◆ Map_Relate()

int32_t Map_Relate ( const void *  value,
MapKey_t  key,
Map *  ctxt 
)

valueをkeyに関連付ける。

同じkeyが既にある場合、関連付けを上書きする。

※　keyとvalueを関連付けるだけであって、valueのスコープと定数/変数は、 ValueFor/ValueAtと合わせ、ユーザーが考慮しなければならない。　※

関連付けできた場合、蓄積済み要素数を返す。

関連付けできなかった場合は0または負。

Parameters
value値。
keyキー。
ctxtコンテキスト。
Returns
蓄積済み要素数。
